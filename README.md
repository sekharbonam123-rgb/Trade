# AI Trading Dashboard

An educational AI-assisted trading dashboard for NIFTY & SENSEX markets.

## 🚀 Features

- **AI Trade Signals**: Real-time BUY/SELL signals with confidence levels
- **Market Sentiment Analysis**: AI-powered sentiment scanner
- **Risk Management**: Automatic risk assessment and position sizing
- **Volatility Tracking**: Real-time volatility index monitoring
- **Live Charts**: TradingView integration for live market data
- **AI Modules**:
  - AI Sentiment Scanner
  - Open Interest Analyzer
  - VWAP Trend Engine
  - Smart Money Detector
  - Volatility Predictor
  - Risk Management AI

## 📋 Tech Stack

- **Frontend**: React + Next.js + TypeScript
- **Styling**: Tailwind CSS
- **AI**: OpenAI API
- **Charts**: TradingView Widgets
- **Deployment**: Vercel
- **Broker Integration**: Zerodha/Upstox APIs
- **Notifications**: Telegram Bot

## 🛠️ Installation

1. Clone the repository:
```bash
git clone https://github.com/sekharbonam123-rgb/Trade.git
cd Trade
```

2. Install dependencies:
```bash
npm install
```

3. Set up environment variables:
```bash
cp .env.local.example .env.local
# Edit .env.local with your API keys
```

4. Run the development server:
```bash
npm run dev
```

5. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🚀 Deployment

1. Push to GitHub
2. Connect to [Vercel](https://vercel.com)
3. Add environment variables in Vercel dashboard
4. Deploy
